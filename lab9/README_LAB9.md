## Installation

Install the require Libraries:

1. Install the `faiss` library:

```bash
pip install faiss
```

2. Install the `ollama` library:

```bash
pip install ollama
```

3. Install the `PyPDF2` library:

```bash
pip install PyPDF2
```

4. Install the `flask` library:

```bash
pip install flask
```

5. Install the `dotenv` library:

